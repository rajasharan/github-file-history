# Github File History Viewer
Navigate back & forth thru file history of github projects

## [Demo Link](https://rajasharan.github.io/github-file-history/)


## Dev setup
```sh
$ npm install -g angular-cli
$ ng serve
...

$ ng build
...

$ ng build --prod
...

$ ng github-pages:deploy
...
```
